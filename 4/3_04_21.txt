1. Разработать программу, которая позволяет выравнивать код программы на языках C#, C++, так, чтобы он выглядел
более читаемым (добавляет переходы на новую строке и отступы там, где это необходимо)

Пример:

using System;
	class Main{
static void Main()
{ Console.Write("Hello"); }}

using System;
class Main
{
	static void Main()
	{
		Console.Write("Hello");
	}
}
